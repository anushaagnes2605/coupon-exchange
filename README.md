# Coupon Marketplace App

## 🚀 Idea
A platform where users can sell unused coupon codes and earn money.

## ✨ Features
- Buy & sell coupon codes
- User authentication
- Secure transactions
- Rating system

## 🛠️ Tech Stack
- Frontend: (React / HTML / etc)
- Backend: (Node / Firebase etc)

## 📸 UI Design
Designed using Figma

## ▶️ How to Run
1. Clone the repo
2. Run npm install
3. Start the server
